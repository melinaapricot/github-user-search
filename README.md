# GitHub User Search
React + TypeScript app for looking up GitHub users. Enter a username and search - searching is automatic with debouncing or by using the search button.

Deployed link: https://melinas-github-user-search.netlify.app/

## Features
- Search for GitHub users by username
- View user info (avatar, description, date joined, link)
- List of public repos with pagination
- Light/dark theme toggle
- Debounced input
- Responsive layout

## Tech Stack
- React
- TypeScript
- SCSS
- GitHub REST API
- Vite

## Getting Started

### Install dependencies

`npm install`

### Start dev server
`npm run dev`

## Notes
- Global styles are located in src/styles.
- Each component has it's own .scss file.
- Project aliases (@components, @hooks, @styles) are set up in both tsconfig.json and vite.config.ts.
